# 小米温度与 CORE/温度计对比统计报告

## 1. 数据与统计口径

- 数据来源：`combined_temperature_test_data.csv`，由 `人体温度传感器测试.xlsx` 清洗整合而来；共 107 条有效点测记录，4 名受试者。
- 伪金标：`CORE_Chest_CBT`（CORE 胸部核心温度）。同时保留 `CORE_Chest_Skin` / `CORE_Wrist_Skin` 作为皮温参考。
- 对比设备：小米温度、水银温度计、CORE 腕部体温。CORE 腕部体温在全天场景中缺失，因此样本量少于小米/水银。
- 场景归类：运动挑战；24 小时生活趋势（办公、娱乐、通勤、进食、躺床、摘表等）；衣着/遮挡环境干扰（薄/厚外套、裸露/遮挡）。
- 数据质量校正：原始 combined 文件中，场景123登记表的 75 条记录只有时分秒，被整合脚本统一补成 2026-03-31；本报告按连续 CORE 文件所在日期恢复为 `Time_Corrected`，原始表不被覆盖。

## 2. 结论摘要

- 小米相对 CORE 胸部核心温度的总体偏差为 -0.27 ℃，MAE 为 0.47 ℃，RMSE 为 0.60 ℃；只有 30% 的点落在 ±0.2 ℃ 内，45% 落在 ±0.3 ℃ 内。
- 小米的总体相关性不足（Pearson r=-0.19），相邻时间点趋势也弱：全场景 delta 相关约 -0.04；当 CORE 真的发生非零变化时，小米同向变化比例只有 17%。
- 最差风险来自个体和场景耦合。李芷欣的总体 MAE 为 0.80 ℃；她在衣着/遮挡场景中 MAE 接近 1 ℃，最高误差 1.5 ℃。
- 运动场景的绝对误差看起来较小（MAE 0.24 ℃），但这不能证明可追踪动态趋势：潘亚勇运动挑战中小米全程为 37.2 ℃，而 CORE 胸部核心温度仍有约 0.4 ℃ 的变化。

## 8. 高误差状态与可能原因

### 8.1 最容易不准确的状态

- 个体基线/佩戴状态差异：同一算法在不同人之间偏差方向不同，李芷欣显著低估，潘亚勇部分场景接近或偏高。未来必须做个人基线校准。
- 衣着、遮挡、局部热环境变化：衣着/遮挡类 MAE 最高，尤其李芷欣厚外套遮挡时 CORE 胸部核心温升至 37.6-38.0 ℃，小米仍停在 36.3-36.5 ℃。
- 核心温快速变化或状态切换：当上一段 CORE 核心温变化 ≥0.2 ℃ 时，小米当前点 MAE 更高；趋势方向也常不一致。
- 夜间/洗澡/摘戴/重新佩戴前后：24 小时数据中有跨夜、洗澡后、躺床等转场；小米可能出现不变、突降或与 CORE 方向不同。
- 运动/出汗：本数据中运动的绝对误差不算最大，但小米常平台化，不能可靠描述运动中核心温变化方向。
- 有风：原始表没有显式“风/风速”字段，因此不能直接定量证明。可作为后续重点实验；当前只能从通勤、裸露等暴露场景看到误差偏大的迹象。

## 9. 统计限制与下一轮验证建议

- 样本量仍小：4 人、107 个点；只有 1 名女性，且没有覆盖真实月经周期。
- 缺少多晚重复睡眠数据，因此夜间恢复/多日绝对值差异目前证据不足。
- 缺少风速、室温、湿度、佩戴松紧、手腕暴露程度、洗澡/摘戴精确事件等环境标签。
- 建议下一轮实验固定佩戴位置和松紧，记录室温/风/衣着/活动强度；每人至少连续 14-28 天夜间同一时间窗口采样，并用个人基线、移动平均和异常状态剔除后再评估经期/恢复/昼夜节律模型。
